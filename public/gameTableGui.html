Games Table

GET /games

Gets all games in the database by gameId, gameName, userId, and courseId

GET /game/:gameId

Gets a single game by the gameId inside the path parameter

POST /game

Creates a game by inserting gameId, and gameName into the database, userId, and courseId come from two other tables

DELETE /game/:gameId

Deletes a game by the gameId

PUT /game/:gameId

Updates games inside the Games table, using gameId, gameName, userId, and courseId which again userId, and courseId are coming from other tables.
