Users Table

GET /users

Gets all users in the database by userId, username, and passwordHash

GET /user/:userId

Gets a single user by his/ her userId inside the path parameter

POST /user

Creates a user, using the parameters: userId, username, and passwordHash

DELETE /user/:userId

Deletes a user by his / her userId

PUT /users/:userId

Updates user inside the Users' table by either their userId, username, and/or passwordHash all at once or individually that is up to the client.
