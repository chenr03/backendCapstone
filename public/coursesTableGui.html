Courses Table

GET /courses

Gets all courses in the Courses Table in the database by courseId, courseName, Hole1, Hole2, Hole3, Hole4, Hole5, Hole6, Hole7, Hole8, Hole9, Hole10, Hole11, Hole12, Hole13, Hole14, Hole15, Hole16, Hole17, Hole18.

GET /course/:courseId

Gets a single course by its courseId inside the path parameter

POST /course

Creates a course by inserting into the table all of these path params: courseId, courseName Hole1, Hole2, Hole3, Hole4, Hole5, Hole6, Hole7, Hole8, Hole9, Hole10, Hole11, Hole12, Hole13, Hole14, Hole15, Hole16, Hole17, Hole18.

DELETE /course/:courseId

Deletes a course by its courseId

PUT /course/:courseId

Updates course inside the Courses' table by updating each one individually or all at once courseId, courseName, Hole1, Hole2, Hole3, Hole4, Hole5, Hole6, Hole7, Hole8, Hole9, Hole10, Hole11, Hole12, Hole13, Hole14, Hole15, Hole16, Hole17, Hole18.
